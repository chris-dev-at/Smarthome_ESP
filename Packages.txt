Packages:

1.	ADD_DEV			(C# -> Arduino)
		-> Adds Device (dev_id, Pin, dev_type)
	
2.	RM_DEV			(C# -> Arduino)
		-> Removes Device (dev_id)
		   
3.	ACTION			(C# -> Arduino)
		-> Change State, Change LED Color, ... (dev_id, action, custom_action_information)

4. 	STATUS_REPORT
		-> Returns Result of Actions


ADD_DEV = 0
RM_DEV = 1
ACTION = 2
STATUS_REPORT = 3

Device_Type:
	0 	Dummy Device
	1	Relay

Screnario:



0;4;8;1 			PacketType;Device_ID;Pin;Device_Type 			//Adds Device with ID 4 on Pin8, Type: Relay		
1;4				PacketType;Device_ID						//Removes Device 4
2;4;state;1		PacketType;Device_ID;Action;Action_Value		//Turns On Device 4
3;FeedBack_Message	PacketType;Message							//Sends Feedback on what just happened
